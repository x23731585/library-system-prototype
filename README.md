# library-system-prototype
Java-based library managment system prototype developed using NetBeans and JFrame UI. #le books.


The Prototype Allows User to -
Log in to system using valid member credentials.
Search for books by title.
view all available books.
Borrow Selected books, which updates availability status.

Simple memory approach, data is stored locally temporarily while the application is running no database is used.

The GUI 
Login UI - handles users login
MainUI - allows users to access library functions eg search book.


Open Apache-NetBeans 
Select File 
Open Project
Locate LoginUI and Run it or Run whole program.

How to Use the Prototype

Login
    •    Enter a valid username and password on the login screen.
    •    Click Submit.
    •    If the details are correct, the system opens the main library screen.

Search Book
    •    Enter a book title in the search field.
    •    Click Search.
    •    Matching books will be displayed in the table.

Search All
    •    Click Search All to display all books in the system.

Borrow Book
    •    Select a book from the table.
    •    Click Borrow Selected.
    •    If the book is available, its status will update.
    •    If the book has already been borrowed, the system will show that it is not available.
